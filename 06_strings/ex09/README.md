# Exercise 9: Vowels and Consonants

## Description
Count the total number of vowels (A, E, I, O, U, and their lowercase counterparts) and consonants in a given string.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "Programming In C is fun.";
```

### Expected Output
```text
String: Programming In C is fun.
Total Vowels: 6
Total Consonants: 13
```
