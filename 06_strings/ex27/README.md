# Exercise 27: Highest Frequency Character

## Description
Find and print the character that appears most frequently in a given string.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "successfully";
```

### Expected Output
```text
String: successfully
Highest frequency character: 's' (Count: 3)
```
