# Exercise 13: Last Occurrence

## Description
Find the index (position) of the last occurrence of a specific character (provided by the user) within a given string.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "programming";
char target = 'g';
```

### Expected Output
```text
Last occurrence of 'g' found at index: 10
```
