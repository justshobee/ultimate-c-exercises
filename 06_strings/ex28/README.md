# Exercise 28: String to Integer (atoi)

## Description
Implement a function to convert a string of digits into its corresponding integer value (e.g., “123” to 123) without using the built-in atoi(). The function should handle positive numbers and optionally, a leading sign.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str1[] = "123";
char str2[] = "-456";
```

### Expected Output
```text
123 -> 123
-456 -> -456
```
