# Exercise 8: Toggle Case

## Description
Toggle the case of every alphabet character in a string: convert uppercase characters to lowercase, and lowercase characters to uppercase.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "HeLlO wOrLd 123";
```

### Expected Output
```text
Toggled string: hElLo WoRlD 123
```
