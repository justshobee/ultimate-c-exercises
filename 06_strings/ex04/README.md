# Exercise 4: Concatenate Strings

## Description
Join two strings (str1 and str2) together, storing the result in the first string, without using the built-in strcat() function.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str1[50] = "Hello "; 
char str2[] = "World!";
```

### Expected Output
```text
Concatenated string: Hello World!
```
