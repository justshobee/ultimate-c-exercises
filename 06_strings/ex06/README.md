# Exercise 6: Convert to Uppercase

## Description
Convert all lowercase characters in a given string to their corresponding uppercase characters. Leave all other characters (uppercase, digits, symbols) unchanged.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "c Programming Is Fun 123";
```

### Expected Output
```text
Uppercase string: C PROGRAMMING IS FUN 123
```
