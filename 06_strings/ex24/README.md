# Exercise 24: Remove Duplicates

## Description
Remove duplicate characters from a string, keeping only the first occurrence of each character. The operation should be performed in place.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "programmingisgreat";
```

### Expected Output
```text
Original: programmingisgreat
No duplicates: progaminset
```
