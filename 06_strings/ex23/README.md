# Exercise 23: Reverse Words

## Description
Reverse the order of words in a sentence while keeping the characters within each word in their original order (e.g., “C language programming” becomes “programming language C”).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "C programming is fun";
```

### Expected Output
```text
Original: C programming is fun
Reversed words: fun is programming C
```
