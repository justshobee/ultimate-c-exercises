# Exercise 17: Remove Extra Spaces

## Description
Modify a string in place to remove all leading spaces, trailing spaces, and reduce multiple consecutive spaces between words to just a single space.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "  This  is   a  test.   ";
```

### Expected Output
```text
Normalized string: "This is a test."
```
