# Exercise 12: First Occurrence

## Description
Find the index (position) of the first occurrence of a specific character (provided by the user) within a given string. If the character is not found, report that as well.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "programming";
char target = 'g';
```

### Expected Output
```text
First occurrence of 'g' found at index: 3
```
