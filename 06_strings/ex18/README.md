# Exercise 18: Check For Digit

## Description
Determine if a string consists entirely of digit characters (‘0’ through ‘9’). The string must contain at least one character.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str1[] = "12345";
char str2[] = "123a45";
```

### Expected Output
```text
String: "12345"
Result: Contains only digits.
```
