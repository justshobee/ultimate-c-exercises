# Exercise 26: Sort String

## Description
Sort the characters of a string alphabetically (e.g., “Jessa” becomes “Jaess”).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "Jessa";
```

### Expected Output
```text
Original: Jessa
Sorted: Jaess
```
