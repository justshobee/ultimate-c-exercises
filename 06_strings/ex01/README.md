# Exercise 1: Read and Print String

## Description
Write a C program to prompt the user to enter a string (which may contain spaces) and then print the exact string back to the console.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
Enter a string (up to 99 characters):
PYnative
You entered: PYnative
```
