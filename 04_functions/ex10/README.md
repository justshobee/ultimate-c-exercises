# Exercise 10: Global Variable and Scope

## Description
Define a global variable int global_counter = 0; outside of all functions. Write a function update_global_counter() that increments it by 1 and prints its value. Demonstrate how functions can modify global variables and how both main and the function can access it.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Expected Output
```text
In main: Initial global counter = 0
Inside function: Global counter updated to 1
Inside function: Global counter updated to 2
In main: Final global counter = 2
```
