# Exercise 2: Function with Parameters

## Description
Create a function add_numbers() that takes two integers as parameters, calculates their sum, and prints the result inside the function. Call this function from main with two different numbers.

Sample Output:

The sum of 15 and 7 is: 22
The sum of 100 and 25 is: 125
