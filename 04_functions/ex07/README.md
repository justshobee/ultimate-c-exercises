# Exercise 7: Array as a Parameter

## Description
Create a function find_max(int arr[], int size) that takes an integer array and its size as parameters and returns the largest element found in the array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int numbers[] = {45, 12, 88, 5, 92, 33};
```

### Expected Output
```text
The largest element in the array is: 92
```
