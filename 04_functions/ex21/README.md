# Exercise 21: extern Keyword for Function

## Description
Write a two-file program where the function definition is in one file (helper.c) and it’s called from the main file (main.c) using the extern keyword (typically implied via an included header file) to link the function, demonstrating external linkage.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`
- `solution_helper.c`

## Allowed Functions
- `printf`
