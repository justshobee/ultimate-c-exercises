# Exercise 22: Check for Prime

## Description
Write a C program to determine if a number is a Prime number (a number greater than 1 that is only divisible by 1 and itself). Use a loop and the break statement to optimize the check.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int N = 31;
```

### Expected Output
```text
31 is a PRIME number
```
