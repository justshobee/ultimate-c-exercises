# Exercise 3: Print Numbers 1 to 10

## Description
Write a C program that uses a for-loop to print all integers from 1 up to and including 10, each on a new line.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions

## Examples
### Expected Output
```text
1
2
3
4
...
10
```
