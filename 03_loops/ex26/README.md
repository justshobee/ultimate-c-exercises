# Exercise 26: Pyramid Pattern

## Description
Write a C program to print a full pyramid (isosceles triangle) pattern of asterisks. The pattern must be centered. Decide the number of rows N.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int N = 3;
```

### Expected Output
```text
number of rows (N): 3
  *
 ***
*****
```
