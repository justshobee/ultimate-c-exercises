# Exercise 21: Smallest and Largest

## Description
Write a C program that asks the user for the number of integers (N) they will enter. Then, read those N integers and use a single loop to determine and display the smallest and largest values.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
How many numbers will you enter (N)? 3
Enter the numbers:
123 567 894

Largest number entered: 894
Smallest number entered: 123
```
