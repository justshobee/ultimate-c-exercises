# Exercise 6: Sum of First N Integers

## Description
Write a C program to calculate the sum of all natural numbers from 1 to N using a for loop.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int N = 5;
```

### Expected Output
```text
The sum of numbers from 1 to 5 is: 15
```
