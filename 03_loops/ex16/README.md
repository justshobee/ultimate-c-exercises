# Exercise 16: Power of a Number

## Description
Write a C program to calculate x raised to the power y (xy) where x (base) and y (exponent) are integers input by the user. Do this without using the built-in pow() function, relying only on a loop. Assume y is non-negative.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int base = 5, exponent = 3;
```

### Expected Output
```text
5^3 = 125
```
