# Exercise 18: Right-Angled Triangle (Numbers)

## Description
Write a C program to print a right-angled triangular pattern where each row prints increasing numbers, up to the row number (e.g., Row 3 prints 1 2 3).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int rows = 3;
```

### Expected Output
```text
1
1 2
1 2 3
```
