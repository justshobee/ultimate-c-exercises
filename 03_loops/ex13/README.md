# Exercise 13: Reverse a Number

## Description
Write a C program to reverse a given number (e.g., 1234 becomes 4321).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int num = 1234;
```

### Expected Output
```text
Original number: 1234
Reversed number: 4321
```
