# Exercise 10: Factorial

## Description
Write a C program to calculate and print the factorial of a non-negative integer N (e.g., 5!=5×4×3×2×1=120) using a loop.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int N = 5;
```

### Expected Output
```text
Factorial of 5 is 120.
```
