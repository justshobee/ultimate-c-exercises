# Exercise 23: Print Primes in Range

## Description
Write a C program that uses nested loops to print all the prime numbers within a given range, specified by a start limit and an end limit.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int start = 10, end = 20;
```

### Expected Output
```text
Prime numbers between 10 and 20 are:
11 13 17 19
```
