# Exercise 2: Simple Calculator

## Description
Write a C program that Implement a basic calculator that takes two floating-point numbers and a character operator (+, -, *, /) as input. Use a switch statement to perform the corresponding arithmetic operation and display the result.

Enter two operands: 10 20
10.00 + 20.00 = 30.00
