# Exercise 25: Armstrong Number

## Description
Write a C program to check if an integer is an Armstrong number (a number that is equal to the sum of the cubes of its digits, e.g., 153=13+53+33=1+125+27=153). Note: This definition is for 3-digit numbers. For simplicity, assume a 3-digit input.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int num = 153;
```

### Expected Output
```text
153 is an ARMSTRONG number.
```
