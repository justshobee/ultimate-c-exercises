# Exercise 19: Inverted Right-Angled Triangle (Asterisks)

## Description
Write a C program to print an inverted right-angled triangular pattern of asterisks. Decide the number of rows. (e.g., if N=4: ****,***,**,*).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int rows = 3;
```

### Expected Output
```text
number of rows (N): 3
***
**
*
```
