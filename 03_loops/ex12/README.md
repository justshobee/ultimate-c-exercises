# Exercise 12: Count Digits

## Description
Write a C program that uses a while loop to count the number of digits in a given number (e.g., 1234 has 4 digits).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int num = 12568;
```

### Expected Output
```text
The number of digits in 12568 is: 5
```
