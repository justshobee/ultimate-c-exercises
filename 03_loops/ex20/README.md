# Exercise 20: Print Multiplication Tables

## Description
Write a C program to print multiplication tables for numbers 1 to 10, each from ×1 to ×10.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Expected Output
```text
Table of 1
 1 x  1 =   1
 1 x  2 =   2
 ....
 1 x 10 =  10

Table of 2
 2 x  1 =   2
 2 x  2 =   4
 ...
 2 x 10 =  20
....

Table of 10
10 x  1 =  10
10 x  2 =  20
....
10 x 10 = 100
```
