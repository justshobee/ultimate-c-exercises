# Exercise 14: Check for Palindrome

## Description
Write a C program to determine if a given integer is a Palindrome (a number that reads the same forwards and backwards, e.g., 121 or 9009).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int num = 121;
```

### Expected Output
```text
121 is a PALINDROME number
```
