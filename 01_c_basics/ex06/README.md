# Exercise 6: Swapping Numbers

## Description
Write a C program to read two integer values, A and B,
and then swap their contents so that A holds the original value of B,
and B holds the original value of A. Use a third, temporary variable.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int A, B;
A = 10;
B = 20;
```

### Expected Output
```text
--- Before Swap ---
A = 10, B = 20

--- After Swap ---
A = 20, B = 10
```
