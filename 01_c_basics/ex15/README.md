# Exercise 15: Sum of Natural Numbers

## Description
Write a C program to calculate and print the sum of the first N natural numbers.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int N;
N = 10;
```

### Expected Output
```text
The sum of the first 10 natural numbers is: 55
```
