# Exercise 5: Simple Interest

## Description
Calculate simple interest for a loan, given the Principal amount (P), Rate of interest (R), and Time in years (T) as input.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions

## Examples

```c

rate = 8;
time = 3;
```

```text
Simple Interest is: 240.00
```
