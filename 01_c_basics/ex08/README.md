# Exercise 8: ASCII Value

## Description
Write a C program that prompts the user to enter a single character and prints its corresponding ASCII (American Standard Code for Information Interchange) value.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
Enter a character: A
The character entered is: A
The ASCII value of 'A' is: 65
```
