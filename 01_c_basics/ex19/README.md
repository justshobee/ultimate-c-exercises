# Exercise 19: Factorial of a Number

## Description
Write a C program to calculate the factorial of a given non-negative integer N. (The factorial of N is the product of all positive integers less than or equal to N, denoted N!).

Formula: For a non-negative integer n, the factorial is defined as:

n! = n × (n - 1) × (n - 2) × ... × 1

Example (5!):

0! = 1
1! = 1
3! = 3 × 2 × 1 = 6
5! = 5 × 4 × 3 × 2 × 1 = 120
