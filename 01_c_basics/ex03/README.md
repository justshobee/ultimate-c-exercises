# Exercise 3: Arithmetic Operations

## Description
Write a C program that calculates and prints the sum, difference, product, and quotient (division) of two integers.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int a, b;
a = 10;
b = 20;
```

### Expected Output
```text
Sum: 20 + 10 = 30
Difference: 20 - 10 = 10
Product: 20 * 10 = 200
Quotient: 20 / 10 = 2.00
```
