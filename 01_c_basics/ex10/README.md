# Exercise 10: Find Largest of Three Numbers

## Description
Write a C program to find and print the largest of given three numbers (A, B, and C).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int A, B, C;
A = 10;
B = 30;
C = 20;
```

### Expected Output
```text
B (30) is the largest number.
```
