# Exercise 39: Array of Structures

## Description
Using the Student structure from Exercise 34, write a program to store the records of three students. Use a loop to read the data for all three and then display their records.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
--- Enter Data for Student 1 ---
Name: Jessa
Roll Number: 25
Marks: 85

--- Enter Data for Student 2 ---
Name: Jon
Roll Number: 26
Marks: 90

--- Enter Data for Student 3 ---
Name: Gari
Roll Number: 27
Marks: 93

--- All Student Records ---
Student 1: Name=Jessa, Roll=25, Marks=85.00
Student 2: Name=Jon, Roll=26, Marks=90.00
Student 3: Name=Gari, Roll=27, Marks=93.00
```
