# Exercise 7: Swapping Without Third Variable

## Description
Write a C program swap two integer values, A and B, without using a third temporary variable.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int A, B;
A = 10;
B = 20;
```

### Expected Output
```text
--- Before Swap ---
A = 10, B = 20

--- After Swap ---
A = 20, B = 10
```
