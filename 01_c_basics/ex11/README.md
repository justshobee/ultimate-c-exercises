# Exercise 11: Leap Year Check

## Description
Write a C program to determine if a given year is a leap year.

Leap year: A year is a leap year if it is divisible by 400, OR if it is divisible by 4 but NOT by 100. Use the modulo operator (%) and the logical OR (||) and logical AND (&&) operators.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int year;
year = 2024;
```

### Expected Output
```text
2024 is a LEAP YEAR.
```
