# Exercise 21: Reverse a Number

## Description
Write a C program to reverse a given integer (e.g., 1234 becomes 4321).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int num = 123
```

### Expected Output
```text
The reverse of 123 is: 321
```
