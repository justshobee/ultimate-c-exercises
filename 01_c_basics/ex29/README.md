# Exercise 29: Search Element

## Description
Write a C program that takes an integer array and a search value (key) from the user. Check if the key is present in the array. If found, print its index; otherwise, print a “Not found” message.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int arr[5] = {10, 25, 30, 45, 50};
```

### Expected Output
```text
Enter the element to search: 30
Element 30 found at index 2.
```
