# Exercise 27: Sum of Array Elements

## Description
Write a C program to calculate and print the sum of all elements in a pre-initialized integer array of size 5.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[5] = {10, 5, 20, 15, 30};
```

### Expected Output
```text
The elements are: 10, 5, 20, 15, 30
The sum of all array elements is: 80
```
