# Exercise 28: Find Maximum and Minimum

## Description
Write a C program to find and print both the largest (maximum) and smallest (minimum) elements in a given integer array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[6] = {55, 12, 89, 7, 42, 60};
```

### Expected Output
```text
Array elements: {55, 12, 89, 7, 42, 60}
Maximum element: 89
Minimum element: 7
```
