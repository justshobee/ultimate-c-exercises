# Exercise 24: Fibonacci Series

## Description
Write a C program to print the first N terms of the Fibonacci series. (The series starts with 0, 1 and each subsequent number is the sum of the two preceding ones: 0,1,1,2,3,5,8,…).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int N = 8;
```

### Expected Output
```text
Fibonacci Series up to 8 terms:
0, 1, 1, 2, 3, 5, 8, 13,
```
