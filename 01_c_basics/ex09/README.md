# Exercise 9: Check Even or Odd

## Description
Write a C program to determines whether given integer is even or odd.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int num;
num = 12;
```

### Expected Output
```text
12 is an EVEN number.
```
