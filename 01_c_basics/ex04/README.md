# Exercise 4: Circumference/Area of a Circle

## Description
Write a C program to calculate both the area and circumference of a circle, given its radius. Use a pre-defined constant for π with value 3.14159.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
float radius;
radius = 5;
```

### Expected Output
```text
Area of the circle: 78.540
Circumference of the circle: 31.416
```
