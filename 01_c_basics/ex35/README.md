# Exercise 35: Words in a String

## Description
Write a C program to count the number of words in a given string (sentence). Assume words are separated by one or more spaces, and the string may start or end with spaces.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "This is a test string";
```

### Expected Output
```text
The string is: "This is a test string"
Total number of words: 5
```
