# Exercise 48: Copy Array

## Description
Write a C program to declare two arrays, A and B, and copy every element from array A into the corresponding position in array B.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int source_arr[5] = {1, 2, 3, 4, 5};
int dest_arr[5]; // Destination array
```

### Expected Output
```text
Source array elements: 1 2 3 4 5
Destination array elements (after copy): 1 2 3 4 5
```
