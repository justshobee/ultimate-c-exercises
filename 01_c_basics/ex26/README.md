# Exercise 26: Input and Print Array

## Description
Write a C program that prompts the user to enter 5 integer values, stores them in an array, and then prints all the stored elements.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
Enter 5 integers:
Enter element 1: 10
Enter element 2: 20
Enter element 3: 30
Enter element 4: 40
Enter element 5: 50

Elements stored in the array: 10 20 30 40 50
```
