# Exercise 1: Float Input/Output

## Description
Write a C program that takes a floating-point number (a decimal number,
like 3.14159) from the user and prints it, formatted to display with only two decimal places.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
Enter a floating-point number (e.g., 3.14159): 75.35678
The number rounded to two decimal places is: 75.36
```
