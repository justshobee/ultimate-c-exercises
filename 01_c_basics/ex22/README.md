# Exercise 22: Check Palindrome

## Description
Write a C program to check if a given integer is a palindrome (reads the same forwards and backward, e.g., 121,3553).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
int num = 121
```

### Expected Output
```text
121 is a PALINDROME
```
