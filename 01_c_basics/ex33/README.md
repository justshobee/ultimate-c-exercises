# Exercise 33: Copy String

## Description
Write a C program to copy the contents of one string (source) to another string (destination) manually without using the built-in strcpy() function.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char source[] = "Copy Me!";
// Destination must be large enough to hold the source string + '\0'
char destination[20];
```

### Expected Output
```text
Source: Copy Me!
Destination: Copy Me!
```
