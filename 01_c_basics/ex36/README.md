# Exercise 36: String Reversal

## Description
Write a C program to reverse a given string.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "Elon";
```

### Expected Output
```text
Enter a string to reverse: Elon
Reversed string: nolE
```
