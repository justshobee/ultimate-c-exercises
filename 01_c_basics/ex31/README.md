# Exercise 31: Function to Find Maximum in Array

## Description
Write a C function named find_max that takes an integer array and its size as parameters, and returns the largest element in the array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int data[] = {8, 15, 2, 70, 9, 33};
```

### Expected Output
```text
The array elements are: {8, 15, 2, 70, 9, 33}
The largest element found by the function is: 70
```
