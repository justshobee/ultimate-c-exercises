# Exercise 44: Function with Pointer (Pass by Reference)

## Description
Write a function named square_value that takes a pointer to an integer as its only argument. The function should calculate the square of the integer the pointer points to and store the result back into the same memory location.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int x = 5;
```

### Expected Output
```text
Original value of x: 5
Value of x after function call: 25
```
