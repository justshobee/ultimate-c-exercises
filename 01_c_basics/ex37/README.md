# Exercise 37: Character Frequency Count

## Description
Write a C program to count the frequency of each unique character (case-insensitive) in a given string.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "Jessa";
```

### Expected Output
```text
Enter a string: Jessa

Character Frequencies:
a: 1
e: 1
j: 1
s: 2
```
