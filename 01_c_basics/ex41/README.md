# Exercise 41: Pointers to Structures

## Description
Define a Car structure (members: model_year, color). Declare a variable of this structure and a pointer to this structure. Use the structure pointer to: 1) assign values to the members, and 2) print the assigned values.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
Enter Car Model Year: 2024
Enter Car Color: Red

--- Car Details (via Pointer) ---
Model Year: 2024
Color: Red
```
