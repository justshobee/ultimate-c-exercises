# Exercise 20: Count Digits

## Description
Write a C program to counts the total number of digits in a given number.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Given
```c
long long num = 3456;
```

### Expected Output
```text
The number 3456 has 4 digits.
```
