# Exercise 32: String Length Calculation

## Description
Develop a C program that prompts the user to enter a string (a sequence of characters) and then calculates and prints the length of that string without using the built-in strlen() function.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
Enter a string: PYnative
The length of the string is: 8
```
