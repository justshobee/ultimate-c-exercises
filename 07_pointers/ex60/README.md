# Exercise 30: Rotate Array Right by n Positions

## Description
Write a C program to perform a right rotation of an array by an arbitrary number of positions, k.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {1, 2, 3, 4, 5, 6, 7};
int k = 3; // Rotate right by 3 positions
```

### Expected Output
```text
Original array: 1 2 3 4 5 6 7
Array after right rotation by 3: 5 6 7 1 2 3 4
```
