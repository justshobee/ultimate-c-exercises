# Exercise 4: Count Even and Odd Elements

## Description
Write a C program that counts and print the total number of even numbers and the total number of odd numbers stored within an integer array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[8] = {1, 4, 7, 12, 15, 20, 3, 10};
```

### Expected Output
```text
Array: 1, 4, 7, 12, 15, 20, 3, 10
Total even elements: 4
Total odd elements: 4
```
