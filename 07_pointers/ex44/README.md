# Exercise 14: Print Unique Elements

## Description
Develop a C program that initializes an integer array and identifies and prints all unique elements (those that appear exactly once in the array).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {5, 2, 8, 5, 1, 2, 9, 8, 4};
```

### Expected Output
```text
Array: 5 2 8 5 1 2 9 8 4 
Unique elements are: 1 9 4
```
