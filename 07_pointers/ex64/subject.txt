Exercise 34: Find a Missing Number in a Sequence of 1 to N

Practice Problem: Given an array of size N-1 containing distinct integers in the range 1 to N, write a C program to find the single integer that is missing from the sequence.

Given:

// Array contains numbers from 1 to 9, so N=10. Missing is 6.
int arr[] = {1, 2, 3, 4, 5, 7, 8, 9, 10};

Expected Output:

The missing number in the sequence 1 to 10 is: 6
