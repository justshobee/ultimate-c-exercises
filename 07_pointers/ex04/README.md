# Exercise 4: Pointer Size

## Description
Write a program to print the size (in bytes) of a pointer to an integer, a pointer to a character, and a pointer to a float on your system.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Expected Output
```text
Size of int pointer (int *): 8 bytes
Size of char pointer (char *): 8 bytes
Size of float pointer (float *): 8 bytes

Size of 'int' variable: 4 bytes
Size of 'float' variable: 4 bytes
```
