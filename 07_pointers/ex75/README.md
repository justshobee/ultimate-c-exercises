# Exercise 5: Compare Strings

## Description
Check if two strings (str1 and str2) are identical (case-sensitive) and determine the result without using the built-in strcmp() function.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str1[] = "apple";
char str2[] = "apple";
```

### Expected Output
```text
The strings are identical.
```
