# Exercise 20: Search Substring

## Description
Determine if a smaller string (needle) exists anywhere within a larger string (haystack). If found, report the starting index of its first occurrence.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char haystack[] = "This is a search test";
char needle[] = "search";
```

### Expected Output
```text
Haystack: "This is a search test"
Needle: "search"
Substring found at index: 10
```
