# Exercise 16: Non-Repeating Character

## Description
Find and print the first non-repeating character in a given string. If all characters repeat, report that no unique character was found.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "aabbccdeeff";
```

### Expected Output
```text
First non-repeating character: 'd'
```
