# Exercise 2: Accessing Value (Dereferencing)

## Description
Write a program that uses the dereference operator (*) to change the value of a variable through its pointer.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int num = 100;
```

### Expected Output
```text
Original value of num: 100
New value of num after pointer assignment: 500
```
