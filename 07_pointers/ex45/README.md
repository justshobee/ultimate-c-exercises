# Exercise 15: Merge Two Arrays

## Description
Write a C program that takes two separate integer arrays, A and B, and merges all their elements into a third, larger array, C. Print the resulting merged array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arrA[] = {10, 20, 30};
int arrB[] = {40, 50, 60, 70};
```

### Expected Output
```text
Merged array C: 10 20 30 40 50 60 70
```
