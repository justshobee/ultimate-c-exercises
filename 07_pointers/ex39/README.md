# Exercise 9: Find Second Largest and Second Smallest Element

## Description
Write a C program to identify and print the second largest and the second smallest unique elements in a given array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int size = 7;
int arr[7] = {10, 50, 20, 80, 40, 90, 30};
```

### Expected Output
```text
Array elements: 10, 50, 20, 80, 40, 90, 30
Second Largest (Max): 80
Second Smallest (Min): 20
```
