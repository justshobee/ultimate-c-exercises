# Exercise 6: Array Traversal

## Description
Use pointer arithmetic to traverse an integer array and print all its elements.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {10, 20, 30, 40, 50};
```

### Expected Output
```text
Array elements using pointer arithmetic:
Element 0: 10
Element 1: 20
Element 2: 30
Element 3: 40
Element 4: 50
```
