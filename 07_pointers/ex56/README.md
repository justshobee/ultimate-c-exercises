# Exercise 26: Separate Even and Odd Numbers

## Description
Write a C program that takes one integer array and separates its elements into two new arrays: one containing all the even numbers and the other containing all the odd numbers. Print all three arrays.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {10, 15, 22, 27, 34, 41, 50};
```

### Expected Output
```text
Even array: 10 22 34 50 
Odd array: 15 27 41
```
