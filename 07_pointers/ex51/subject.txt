Exercise 21: Merge Two Sorted Arrays

Practice Problem: Given two arrays, A and B, which are both individually sorted in ascending order, write a C program to merge them into a third array, C, such that C is also sorted, without using a sorting algorithm on C.

Given:

int arr1[] = {10, 30, 50, 70};
int size1 = 4;
int arr2[] = {20, 40, 60, 80, 90};
int size2 = 5;

Expected Output:

Merged sorted array: 10 20 30 40 50 60 70 80 90
