Exercise 29: Integer to String (itoa)
Practice Problem: Implement a function to convert an integer to its string representation (e.g., 123 to “123”) without using the built-in sprintf() or itoa(). The function should handle positive and negative integers.

Given:

int num1 = 12345;
int num2 = -987;
Expected Output:

12345 -> "12345"
-987 -> "987"
