Exercise 5: Reverse An Array In Place

Practice Problem: Write a C program to reverse the order of elements in an array without using a second, auxiliary array. The reversal must be performed directly within the original array (in-place).

Given:

int arr[7] = {10, 20, 30, 40, 50, 60, 70};

Expected Output:

Original array: 10 20 30 40 50 60 70 
Reversed array: 70 60 50 40 30 20 10
