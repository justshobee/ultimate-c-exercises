# Exercise 25: Check Anagrams

## Description
Determine if two strings are anagrams of each other (meaning they contain the exact same characters with the same frequencies, regardless of order). Case and spaces should be ignored for a robust check.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str1[] = "listen";
char str2[] = "silent";
```

### Expected Output
```text
String 1: listen
String 2: silent
Result: They are anagrams.
```
