# Exercise 2: String Length

## Description
Calculate and print the length of a string without using the built-in strlen() function.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "Hello World!";
```

### Expected Output
```text
The string is: "Hello World!"
The length of the string is: 12
```
