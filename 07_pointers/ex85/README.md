# Exercise 15: Remove Specific Character

## Description
Remove all occurrences of a specific character (e.g., ‘x’) from a string and shift the remaining characters to fill the gap.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "eExxammpelle";
char char_to_remove = 'e';
```

### Expected Output
```text
String after removing 'e': Exxammpll
```
