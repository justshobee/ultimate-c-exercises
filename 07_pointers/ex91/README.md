# Exercise 21: Reverse String

## Description
Reverse the order of characters in a given string in place (without using an auxiliary string).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "programming";
```

### Expected Output
```text
Original string: programming
Reversed string: gnimmargorp
```
