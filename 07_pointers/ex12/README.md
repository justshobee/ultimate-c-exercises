# Exercise 12: String Copy

## Description
Implement a function custom_strcpy that copies the content of a source string (src) to a destination string (dest) using only character pointers.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char source[] = "Pointer Mastery";
char destination[50]; // Ensure destination buffer is large enough
```

### Expected Output
```text
Source: Pointer Mastery
Destination: Pointer Mastery
```
