# Exercise 7: Convert to Lowercase

## Description
Convert all uppercase characters in a given string to their corresponding lowercase characters. Leave all other characters (lowercase, digits, symbols) unchanged.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "C PrOgRaMmInG Is AwEsOmE";
```

### Expected Output
```text
Lowercase string: c programming is awesome
```
