# Exercise 31: Remove Duplicate Elements from a Sorted Array

## Description
Write a C program to remove duplicate elements from a given array that is already sorted. The modification should happen in-place (i.e., modify the original array), and the function should return the length of the new array with unique elements.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {1, 1, 2, 2, 2, 3, 4, 4, 5};
```

### Expected Output
```text
Original array: 1 1 2 2 2 3 4 4 5
Array after removing duplicates (new size 5): 1 2 3 4 5
```
