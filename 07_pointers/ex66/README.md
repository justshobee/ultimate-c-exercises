# Exercise 36: Calculate Sum of All Elements in a Matrix

## Description
Write a C program to calculate and print the sum of all integer elements present in a given matrix.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}}; // 2x3 matrix
```

### Expected Output
```text
Sum of all elements in the matrix: 21
```
