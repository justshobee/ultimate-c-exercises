# Exercise 23: Swap Two Numbers

## Description
Write a function swap(int *a, int *b) that swaps the values of two integers using their pointers. Demonstrate the swap in the main function.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int x = 100;
int y = 200;
```

### Expected Output
```text
Before swap: x = 100, y = 200
After swap: x = 200, y = 100
```
