# Exercise 16: Check Array Palindrome

## Description
Create a C program that determines whether an array of integers is a palindrome. An array is a palindrome if it reads the same forwards and backwards.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr1[] = {1, 2, 3, 2, 1}; // Palindrome
int arr2[] = {1, 2, 3, 4, 5}; // Not a palindrome
```

### Expected Output
```text
Array 1 is a palindrome.
Array 2 is NOT a palindrome.
```
