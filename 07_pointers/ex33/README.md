# Exercise 3: Maximum and Minimum element

## Description
Write a C program that determines and prints the largest (maximum) and the smallest (minimum) value among all elements in a given array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[6] = {55, 12, 89, 4, 30, 77};
```

### Expected Output
```text
Maximum element in the array: 89
Minimum element in the array: 4
```
