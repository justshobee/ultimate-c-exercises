# Exercise 38: Sum of Rows and Columns of a Matrix

## Description
Write a C program to calculate and print the sum of elements for each individual row and the sum of elements for each individual column of a matrix.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int matrix[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
```

### Expected Output
```text
Sum of Row 1: 6
Sum of Row 2: 15
Sum of Row 3: 24

Sum of Column 1: 12
Sum of Column 2: 15
Sum of Column 3: 18
```
