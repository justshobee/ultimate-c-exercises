# Exercise 11: Search In Array

## Description
Implement a C program to search for a specific target value within an array. The program should return the index (position) of the first occurrence of the element if found, or a message indicating it was not found.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {10, 50, 30, 70, 80, 20, 90, 40};
int search_element = 70;
```

### Expected Output
```text
Element 70 found at index 3.
```
