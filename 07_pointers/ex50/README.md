# Exercise 20: Insertion Sort

## Description
Write a C program to sort an array in ascending order using the Insertion Sort algorithm.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {12, 11, 13, 5, 6};
```

### Expected Output
```text
Array sorted using Insertion Sort: 5 6 11 12 13
```
