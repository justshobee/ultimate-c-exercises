# Exercise 16: Dynamic Array

## Description
Dynamically allocate memory for an array of 5 integers using malloc(). Read 5 integer values from the user into the array, print them using pointer arithmetic, and finally, free the allocated memory.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `exit`
- `free`
- `malloc`
- `printf`
- `scanf`

## Examples
### Given
```c
int count = 5;
```

### Expected Output
```text
Enter 5 integers:
Element 1: 10
Element 2: 20
Element 3: 30
Element 4: 40
Element 5: 50

Stored elements are: 10 20 30 40 50
```
