# Exercise 14: Character Frequency

## Description
Calculate and print the frequency (count) of every lowercase alphabet character (‘a’ through ‘z’) present in a given string

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
char str[] = "Jessa";
int freq[26] = {0}; // Initialize all 26 counters to zero
```

### Expected Output
```text
Character Frequencies (a-z):
'a': 1
'e': 1
'j': 1
's': 2
```
