# Exercise 22: Remove Duplicate Elements

## Description
Write a C program that removes duplicate elements from an array, modifying the array in-place to contain only unique elements, and returns the new size of the modified array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {10, 20, 20, 30, 10, 40, 50, 50, 40};
```

### Expected Output
```text
Original array (size 9): 10 20 20 30 10 40 50 50 40 
Array after removing duplicates (new size 5): 10 20 30 40 50
```
