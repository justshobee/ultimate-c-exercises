# Exercise 22: 2D Array Simulation

## Description
Simulate a 2×3 2D array using a single pointer to dynamically allocated memory. Allocate memory for 6 integers. Use pointer arithmetic to access and print the value at the conceptual row 1, column 2 (index [1][2]).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `free`
- `malloc`
- `printf`

## Examples
### Given
```c
const int ROWS = 2;
const int COLS = 3;
```

### Expected Output
```text
Simulated 2x3 Array:
Value at index [1][2] (offset 5): 15
```
