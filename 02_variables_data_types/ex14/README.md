# Exercise 14: Octal to Decimal

## Description
Print decimal equivalent of a octal.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int octal_num = 015; // Represents 13 in decimal
```

### Expected Output
```text
The octal number 015 in decimal is: 13
```
