# Exercise 6: Variable Swapping (Third Variable)

## Description
Swap the values of two integer variables using a third temporary variable. Print the values before and after the swap

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int a = 5;
int b = 10;
```

### Expected Output
```text
Before swap: a = 5, b = 10
After swap: a = 10, b = 5
```
