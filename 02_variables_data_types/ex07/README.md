# Exercise 7: Variable Swapping (Without Third Variable)

## Description
Write a C program to swap the values of two integer variables without using a third variable.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int a = 5;
int b = 10;
```

### Expected Output
```text
Before swap: a = 5, b = 10
After swap: a = 10, b = 5
```
