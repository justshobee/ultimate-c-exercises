# Exercise 14: Enumeration (Enum) Basic

## Description
Define an enumeration (enum) named DaysOfWeek containing the seven days of the week, starting from SUNDAY. Explicitly set MONDAY to start at the value 1. Print the integer value associated with each day.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Expected Output
```text
--- Days of the Week Values ---
SUNDAY: 0
MONDAY: 1
TUESDAY: 2
WEDNESDAY: 3
THURSDAY: 4
FRIDAY: 5
SATURDAY: 6
```
