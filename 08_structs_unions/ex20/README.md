# Exercise 20: Passing Array of Structures to a Function

## Description
Define a Student structure (name, percentage). Input data for 4 students. Write a function calculate_class_average that accepts the array of structures and its size, calculates the average percentage of the entire class, and returns the result. Display the average in main

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
struct Student {
    char name[50];
    float percentage;
};
```
