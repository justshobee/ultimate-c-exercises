# Exercise 6: Returning Structure from a Function

## Description
Define a Rectangle structure (members: length, width, area). Write a function, get_rectangle_data, that prompts the user for the length and width, calculates the area, and returns the complete Rectangle structure back to the calling function (main).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`
- `scanf`

## Examples
### Expected Output
```text
Enter Length of Rectangle: 10
Enter Width of Rectangle: 5

--- Rectangle Report ---
Length: 10.00
Width: 5.00
Calculated Area: 50.00
```
