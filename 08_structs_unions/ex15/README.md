# Exercise 15: Enum in Structure

## Description
Define an enum for Color (RED, GREEN, BLUE). Define a Product structure with members: name (string), price (float), and an instance of the Color enumeration called product_color. Input and display the details of a product, using the enum to assign and display the color.
