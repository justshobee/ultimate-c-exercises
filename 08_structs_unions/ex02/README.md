# Exercise 2: Structure Initialization at Declaration

## Description
Define a Book structure with members: title (string), author (string), pages (integer), and price (float). Declare a structure variable and initialize all its members at the time of declaration using an initializer list. Finally, display all the initialized details.

## Expected Files

Pages: 272
Price: $12.50
