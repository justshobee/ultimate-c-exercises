# Exercise 17: Sort An Array

## Description
Write a C program to sort an array of integers in ascending order using the Bubble Sort algorithm.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {64, 34, 25, 12, 22, 11, 90};
```

### Expected Output
```text
Array sorted in ascending order: 11 12 22 25 34 64 90
```
