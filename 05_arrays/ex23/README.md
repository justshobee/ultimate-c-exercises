# Exercise 23: Count Frequency Of Each Element

## Description
Write a C program to determine the frequency (number of occurrences) of every unique element within a given array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {2, 5, 2, 8, 5, 1};
```

### Expected Output
```text
Frequencies:
Element 1: 1 times
Element 2: 2 times
Element 5: 2 times
Element 8: 1 times
```
