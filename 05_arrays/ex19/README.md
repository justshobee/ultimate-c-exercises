# Exercise 19: Sort An Array In Descending Order

## Description
Write a C program to sort an array of integers in non-increasing (descending) order using the Selection Sort algorithm.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {64, 25, 12, 22, 11};
```

### Expected Output
```text
Array sorted in descending order: 64 25 22 12 11
```
