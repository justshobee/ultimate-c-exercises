# Exercise 29: Rotate Array Left by n Positions

## Description
Write a C program to perform a left rotation of an array by an arbitrary number of positions, k.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {1, 2, 3, 4, 5, 6, 7};
```

### Expected Output
```text
Original array: 1 2 3 4 5 6 7
Array after left rotation by 3: 4 5 6 7 1 2 3
```
