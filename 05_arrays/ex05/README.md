# Exercise 5: Reverse An Array In Place

## Description
Write a C program to reverse the order of elements in an array without using a second, auxiliary array. The reversal must be performed directly within the original array (in-place).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[7] = {10, 20, 30, 40, 50, 60, 70};
```

### Expected Output
```text
Original array: 10 20 30 40 50 60 70
Reversed array: 70 60 50 40 30 20 10
```
