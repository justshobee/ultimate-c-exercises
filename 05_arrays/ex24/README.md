# Exercise 24: Most Frequent Element

## Description
Write a C program to find and print the element that appears the greatest number of times (the mode) in a given array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {2, 5, 2, 8, 5, 1, 5, 9, 1, 2, 5};
```

### Expected Output
```text
The most frequent element is: 5
```
