# Exercise 2: Sum and Average

## Description
Write a C program that calculates and prints the total sum and the arithmetic average of all numerical elements present in a given array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[5] = {10, 20, 30, 40, 50};
```

### Expected Output
```text
Sum of array elements: 150
Average of array elements: 30.00
```
