# Exercise 21: Merge Two Sorted Arrays

## Description
Given two arrays, A and B, which are both individually sorted in ascending order, write a C program to merge them into a third array, C, such that C is also sorted, without using a sorting algorithm on C.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr1[] = {10, 30, 50, 70};
int size1 = 4;
int arr2[] = {20, 40, 60, 80, 90};
int size2 = 5;
```

### Expected Output
```text
Merged sorted array: 10 20 30 40 50 60 70 80 90
```
