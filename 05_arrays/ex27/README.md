# Exercise 27: Rotate Array Left by One Position

## Description
Write a C program to perform a left rotation of an array by one position. In a left rotation, every element is shifted one index to the left, and the first element moves to the end of the array.

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {1, 2, 3, 4, 5};
```

### Expected Output
```text
Original array: 1 2 3 4 5
Array after left rotation by 1: 2 3 4 5 1
```
