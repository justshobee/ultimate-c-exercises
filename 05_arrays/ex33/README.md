# Exercise 33: Reverse a Portion of an Array

## Description
Write a C program to reverse the elements of an array between a specified start index and end index (inclusive).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`

## Allowed Functions
- `printf`

## Examples
### Given
```c
int arr[] = {10, 20, 30, 40, 50, 60, 70};
```

### Expected Output
```text
Original array: 10 20 30 40 50 60 70
Array after reversing indices 2 to 5: 10 20 60 50 40 30 70
```
