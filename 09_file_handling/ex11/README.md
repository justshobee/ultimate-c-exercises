# Exercise 11: Uppercase Conversion

## Description
Read the content of any existing text file (e.g. data.txt updated in Exercise 3) and write its content to a new file (output_upper.txt), converting all lowercase letters to uppercase letters. All other characters (numbers, symbols, spaces) should remain unchanged.

## Assignment File
- `solution.c`

## Expected Files
- `data.txt`
- `new_data.txt`
- `solution.c`

## Allowed Functions
- None (Only standard operators/keywords)

## Examples
### Expected Output
```text
output_upper.txt

NAME: ALICE JOHNSON
AGE: 30 YEARS
DATE APPENDED: 2025-10-15
STATUS: SUCCESSFULLY APPENDED.
```
