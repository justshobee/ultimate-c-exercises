# Exercise 17: Tab to Spaces Conversion

## Description
Read a file (tabbed.txt) and replace every tab character (\t) with exactly four space characters (), saving the result to a new file (spaced.txt).

## Assignment File
- `solution.c`

## Expected Files
- `solution.c`
- `spaced.txt`
- `tabbed.txt`

## Allowed Functions
- None (Only standard operators/keywords)

## Examples
### Expected Output
```text
Tabs replaced with 4 spaces. Saved to 'spaced.txt'.
```
