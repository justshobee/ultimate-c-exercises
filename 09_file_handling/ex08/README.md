# Exercise 8: Display a Specific Line of a File

## Description
Prompt the user to enter a line number N. Read a text file (e.g. data.txt updated in Exercise 3) and display the content of the Nth line on the console.

## Assignment File
- `solution.c`

## Expected Files
- `data.txt`
- `solution.c`
- `test.txt`

## Allowed Functions
- None (Only standard operators/keywords)

## Examples
### Expected Output
```text
Enter the line number to display: 2

--- Line 2 ---
Age: 30 years
--------------
```
