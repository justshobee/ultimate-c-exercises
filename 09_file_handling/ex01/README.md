# Exercise 1: Create and Write to a File

## Description
Develop a C program to create a new text file named data.txt and write two lines of personal information (e.g., your name and age) to it using fprintf().

## Assignment File
- `solution.c`

## Expected Files
- `data.txt`
- `solution.c`

## Allowed Functions
- None (Only standard operators/keywords)

## Examples
### Expected Output
```text
Data successfully written to data.txt.

data.txt
Name: Alice Johnson
Age: 30 years
```
