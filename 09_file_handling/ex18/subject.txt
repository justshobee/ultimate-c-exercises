Exercise 18: Simple Encryption (Caesar Cipher)

Practice Problem: Implement a simple Caesar cipher (shift by N=13 or rot13) to encrypt the contents of an input file (plain.txt) and write the result to an output file (encrypted.txt). Only alphabet characters (A-Z, a-z) should be shifted; all others should be written as is.

Given:

plain.txt

Name: Alice Johnson
Age: 30 years

Expected Output:

encrypted.txt

Qdph: Dolfh Mrkqvrq
Djh: 30 bhduv
