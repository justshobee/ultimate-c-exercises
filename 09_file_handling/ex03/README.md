# Exercise 3: Append Data to a File

## Description
Open the existing file data.txt(created in Exercise 1) in append mode and add today’s date and a short message (“Successfully appended.”) as new lines to the end of the file.

## Assignment File
- `solution.c`

## Expected Files
- `data.txt`
- `solution.c`

## Allowed Functions
- None (Only standard operators/keywords)

## Examples
### Expected Output
```text
data.txt

Name: Alice Johnson
Age: 30 years
Date Appended: 2025-10-15
Status: Successfully appended.
```
