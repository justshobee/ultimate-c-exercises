# Exercise 16: Merge Two Files

## Description
Read the content of two separate files (file1.txt and file2.txt) and write their combined content sequentially into a third file (merged.txt).

## Assignment File
- `solution.c`

## Expected Files
- `file1.txt`
- `file2.txt`
- `marge.txt`
- `solution.c`

## Allowed Functions
- None (Only standard operators/keywords)

## Examples
### Given
```text
file1.txt
This is file 1

file2.txt
This is file2
```

### Expected Output
```text
merged.txt

This is file 1
--- End of File 1 ---
This is file 2
```
