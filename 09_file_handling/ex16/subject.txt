Exercise 16: Merge Two Files

Practice Problem: Read the content of two separate files (file1.txt and file2.txt) and write their combined content sequentially into a third file (merged.txt).

Given:

file1.txt
This is file 1

file2.txt
This is file2

Expected Output:

merged.txt

This is file 1
--- End of File 1 ---
This is file 2
